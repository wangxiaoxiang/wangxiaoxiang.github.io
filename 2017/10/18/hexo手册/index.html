常用hexo指令

需要使用终端进入到博客根目录下


然后可执行的操作：

  1. hexo s :本地预览博客
  2. hexo clean :删除目录下的public文件夹
  3. hexo g :生成静态文件夹（public）
  4. hexo g -w :生成静态文件并监视文件变动
  5. hexo d :部署到网站
  6. 详细的官方文档
